Efail in Thunderbird

In the end of 2017 researchers discovered a vulnerability which allowed attackers to reveal encrypted emails on the basis of questionable implementations of security features in various email clients.

In this paper I examined this vulnerability in very detail according to the implementation of the open source email client Mozilla Thunderbird.

kNN in Recommendation Engines

This work deals with the question, how the k-nrearest Neighbors algorithm (kNN) can be used in recommendation engines.

Also it is intended to show the relevance of kNN in recommendation systems. Thereby the practical applicability of the practical applicability of the algorithm in this context in focused. So how it can be used in practice and which problems or questions are currently open in research and in practice.

Malware Analysis

The freedom designed internet allows malicious software to spread incredibly fast and inconspicuously - mostly even highly obfuscated. Therefore the analysis of such software becomes more and more important and necessary for an accurate incident response.

Common tools and techniques are explained here which master the analysis.
