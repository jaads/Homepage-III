Web Components using Lit Element

A simple web component

Web component are made out of four main specifications:

  • ES6 Modules
  • Custom Elements
  • Shadow DOM
  • HTML Template
Web components are a set of web platform APIs that allow you to create new custom, reusable, encapsulated HTML tags to use in web pages and web apps. webcomponents.org

The light grey box below is one single web component consisting of properties, stlyes and functions triggered by buttons.

It ca be placed anywhere on the website by using the custom element as follows:

<my-stopwatch></my-stopwatch>

A navigation element as web component

In this senarios two majar techniques had been used in order to reach as much independenceas of the component as possible. First the fact that each property of a compontent made with Lit-Element is automaticcaly converted to a HTML attribute of the custom elememt and hence editable in a declarative fashion from the HTML file.

Furthermore slots has beed used inject conten into the component. Here is the component:

import { LitElement, html, css } from 'https://unpkg.com/lit-element/lit-element.js?module';
    
    class NavItem extends LitElement {
    
        static get properties() {
            return {
                reference: { type: String }
            };
        }
    
        constructor() {
            super();
            this.reference = '#';
        }
    
        render() {
            return html`
            <a href="${this.reference}"><slot></slot></a>
            `;
        }
    }
    
    customElements.define('my-navitem', NavItem);

And how the above web compoennt can be used is shown below. reference is the propbierty of the component.

<my-navitem reference="../../index.html" class="menuitem navitem">Home</my-navitem>
